# Documentação

Esta área contém a documentação detalhada do CPS Design System, com exemplos da aplicação das instruções do Guia Visual em diversos elementos recorrentes em sistemas e aplicações Web em geral.

Atenção

Esta documentação é puramente teórica e ilustrativa, apresentando apenas exemplos da aplicação das instruções do Guia Visual. Se você procura uma biblioteca de componentes totalmente aderente ao CPS Design System, pronta para uso diretamente em seus projetos de soluções Web, acesse CPS Elements (opens new window).
